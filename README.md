## Set up and Compile

For windows:

- node.js (v 10.22.0)
- npm (v 6.14.6)
- MSVC 2017 v15.0 (MSVC 2019 should be ok)



```shell
>> cd <your dir>
>> npm clean cache --force
>> npm install
```



## Edit and Run

```shell
>> npm run watch
```



Then any changes (in `./scss/style.scss` and `./js/scripts.js`) will be automatically compiled to .css and saved.
